# Списки
Чтобы добавить ненумерованные списки, необходимо пункты выделить звездочкой (*)
например, 
* вот так
* элемент 2
* элемент 3

Чтобы добавить нумерованнные списки необходимо просто добавить номер. Например
1. Первый пункт
2. Второй пункт
